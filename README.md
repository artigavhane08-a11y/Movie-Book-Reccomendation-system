# # Movie & Book Recommendation System

This project is a content-based recommendation system built using Python.

It recommends:
- Movies based on genre similarity
- Books based on title similarity

Technologies Used:
- Python
- Pandas
- Scikit-learn
- Cosine Similarity
- CountVectorizer

Datasets:
- MovieLens Dataset
- Goodreads Books Dataset

Features:
- User can choose between movie or book recommendation
- Provides top 5 similar recommendations
- Handles dataset errors using data preprocessing

How to Run:
1. Place movies.csv and books.csv in the project folder
2. Run: python app.py
3. Enter movie or book name

This project demonstrates practical implementation of Natural Language Processing and Recommendation Systems.
